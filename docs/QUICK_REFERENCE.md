## üîß Emergency Fixes

### Lost/No Tools
```
1. Punch trees for wood
2. Make wooden pickaxe
3. Mine stone for stone pickaxe
4. Continue normal progression
```

### No Food/Starving
```
1. Kill any nearby animals
2. Look for apples under oak trees  
3. Find village for bread/crops
4. Emergency: eat rotten flesh (causes hunger effect)
```

### Surrounded by Mobs
```
1. Build pillar 3 blocks up (mobs can't reach)
2. Wait for daylight (zombies/skeletons burn)  
3. Fight from high ground with bow/sword
4. Dig escape tunnel if needed
```
